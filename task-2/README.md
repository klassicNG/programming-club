# Week 2 Assessment - C++ Programming Club

## Project Structure
- `array.cpp`: Demonstrates array traversal using standard and range-based loops.
- `pointer.cpp`: Shows how to manipulate data using memory addresses.
- `reference.cpp`: Illustrates the use of reference variables as aliases.
- `structure.cpp`: Implements a basic `Rectangle` structure and calculates area.
